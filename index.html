Proyectos

INEDI-ACCES

React Native + Expo • JS • Firebase

Este proyecto consiste en el desarrollo de una plataforma de gestión educativa centralizada diseñada para resolver las problemáticas operativas de una institución educativa mediante la consolidación de procesos académicos y administrativos en un único sistema. La solución implementa un modelo de control de acceso basado en roles que diferencia funcionalmente entre dos tipos de usuarios: los docentes, quienes gestionan sus grupos asignados, las materias que imparten, el registro de calificaciones y la generación de boletas; y los estudiantes, quienes acceden a su carga académica, consultan sus boletas de evaluación y disponen de un espacio para la carga de tareas. Uno de los principales desafíos técnicos abordados fue la implementación de un sistema de permisos granular que restringe y habilita funcionalidades específicas según el perfil de usuario, lo cual representó un punto de inflexión en mi desarrollo profesional como ingeniero de soluciones al exigir el diseño de arquitecturas seguras con separación lógica de privilegios y validación de autorización en cada capa funcional de la aplicación.

Patronfy

React Native + Expo • TypeScript • Vercel

Aplicación móvil orientada a la democratización del patronaje de ropa mediante un modelo freemium. Implementada con TypeScript para garantizar la integridad de los datos antropométricos, la solución automatiza la generación de patrones de vestir personalizados. El desafío principal fue traducir lógica compleja de diseño textil en algoritmos de software escalables y fáciles de usar.

POSINV Creaciones Madriz

Electron • NestJS

Este proyecto representa una de las soluciones más significativas en mi trayectoria, desarrollado en el marco de mi pasantía profesional como un sistema de gestión empresarial integral diseñado bajo especificaciones directas del cliente y orientado a la centralización de procesos administrativos y operativos mediante un dashboard analítico con métricas clave del negocio, un punto de venta para procesamiento de transacciones comerciales, gestión de inventario con clasificación por categorías y control de stock en tiempo real, reportes financieros de estados de cuenta y flujo de caja, y reportes de inventario activo con seguimiento de existencias y rotación de productos, todo implementado con arquitectura monolítica e interfaz unificada que incorpora la paleta cromática corporativa del negocio, tipografía optimizada para legibilidad y patrones de navegación intuitivos orientados al usuario final, cumpliendo así una función dual como solución de software productiva para el cliente y experiencia formativa en desarrollo de sistemas empresariales bajo metodologías de trabajo reales.
